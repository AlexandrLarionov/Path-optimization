# Оптимизация траектории мобильного робота

## Описание проекта

В проекте реализован алгоритм построения и оптимизации траектории мобильного робота в двумерной среде с круговыми препятствиями.

Основная цель — получить гладкую, короткую и безопасную траекторию, минимизируя целевую функцию, учитывающую:

- длину пути,
- кривизну,
- расстояние до препятствий.

---

## Структура алгоритма

Процесс построения траектории включает следующие этапы:

### 1. Построение карты

- Рабочее пространство представляется квадратной областью.
- Препятствия моделируются в виде окружностей.
- Среда дискретизируется в регулярную сетку для применения алгоритма поиска пути.

---

### 2. Поиск пути алгоритмом A*

- Используется алгоритм A* с евклидовой эвристикой.
- Получается дискретная ломаная траектория от старта до цели.
- Разрешены движения по 8 направлениям (включая диагонали).

---

### 3. Параметризация траектории B-сплайном

- Из траектории A* выбирается набор контрольных точек.
- Строится кубический B-сплайн.
- Получается гладкая параметрическая кривая.

---

### 4. Оптимизация траектории

Оптимизируются контрольные точки B-сплайна (кроме стартовой и конечной).

Задача формулируется как:

\[
\min_{\mathbf{P}} J(\mathbf{P})
\]

где целевая функция:

\[
J = w_1 L + w_2 C + w_3 O
\]

- **L** — длина траектории  
- **C** — средняя кривизна  
- **O** — штраф за приближение к препятствиям  

Минимизация выполняется методом **SLSQP** из библиотеки SciPy.

---

## Используемые библиотеки

- Python 3
- NumPy
- SciPy
- Matplotlib


## Результат

В результате оптимизации:

- уменьшается длина пути,
- снижается средняя кривизна,
- увеличивается минимальное расстояние до препятствий,
- достигается локальный минимум целевой функции.
